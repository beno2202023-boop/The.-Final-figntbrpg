# The Final Fight RPG

A Final Fight-inspired RPG game built with Java and LibGDX.

## Description
It's final fight but it's a/rpg - A blend of classic beat-em-up action with RPG mechanics.

## Tech Stack
- **Language:** Java
- **Framework:** LibGDX
- **Build Tool:** Maven

## Project Structure
```
src/
├── main/
│   ├── java/
│   │   └── com/finalfightrpg/

│   │       ├── entities/
│   │       ├── systems/
│   │       └── assets/
│   └── resources/
└── test/
    └── java/
```

## Getting Started

### Prerequisites
- Java 11 or higher
- Maven 3.6+

### Building
```bash
mvn clean package
```

### Running
```bash
mvn exec:java@run
```
